# STANDARDS_OCC

## 1. PADRÃO RELATÓRIO (Reports)

- Cabeçalho: Panel com borda Size: 1.
- Conteúdo do Cabeçalho:
  - Título do Relatório (Centro, Negrito).
  - Nome da Empresa (Dinâmico, Centro).
  - Texto "Emitido em: DD/MM/AAAA" (Centro, Fonte menor).
- Corpo: Tabela com linhas alternadas (Zebrada) para leitura clara.
- Se for por período: "De: DD/MM/AAAA até: DD/MM/AAAA".

## 2. PADRÃO FORMULÁRIO (Forms)

- Header (Top Panel): Nome do formulário centralizado, altura fixa.
- Body (Client Panel): Área central para os campos com margem (Padding) de 20px.
- Footer (Bottom Panel): Botões de ação sempre alinhados à direita:
  - [ Cancelar (Cinza) ]
  - [ Salvar (Cor de Destaque) ]

## 3. PADRÃO DE CORES (Status)

- ATIVO / SUCESSO: Verde.
- TRIAL / INFO: Azul.
- PENDENTE / ALERTA: Laranja/Amarelo.
- CANCELADO / ERRO: Vermelho.

## 4. REGRA DE USO

- Sempre que solicitado "Siga o Padrão X", aplicar estas diretrizes na nova tela.
- Se a tela atual estiver fora deste padrão, ajustar para aderência.
