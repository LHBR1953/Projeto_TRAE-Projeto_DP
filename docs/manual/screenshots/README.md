# Screenshots do Manual

Este diretório guarda as imagens referenciadas pelo manual:

- [Manual_do_Usuario.md](../Manual_do_Usuario.md)

## Como gerar as imagens (modo manual)
1) Abra o sistema em `http://localhost:8282/`
2) Navegue até a tela desejada
3) Use a Ferramenta de Captura do Windows (Snipping Tool)
4) Salve com o nome indicado no manual (ex.: `70-financeiro.png`)

Sugestão:
- Sempre capture a tela inteira do módulo e, se necessário, faça uma segunda captura apenas do modal.
- Evite incluir dados sensíveis de pacientes em prints reais.
