OCC — Guia do Usuário (Administrador da Clínica) 🦷✨

                    
Bem-vindo(a)! Este guia é para você, Administrador(a) da Clínica, que acabou de receber o acesso ao OCC.


                    
1) Primeiro acesso 🔐

                    
    
                        
  1. Abra o OCC no link que você recebeu 🌐
    2. 
                        
  2. Faça login com:
                            
      
                                
    • 📧 E-mail: o e-mail da clínica
      • 
                                
    • 🔑 Senha inicial: a senha enviada pelo responsável do sistema
      • 
                            
    
                        
    3. 
                        
  3. ✅ Ao entrar, você já está no ambiente da sua clínica
    4. 
                    

                    
Dica: troque a senha assim que possível 🔁🛡️


                    
2) Entendendo o menu 🧭

                    
    
                        
  • ⏱️ Dashboard (Indicadores do dia e visão executiva da unidade)
    • 
                        
  • 👤 Pacientes
    • 
                        
  • 👥 Profissionais
    • 
                        
  • 🧰 Serviços
    • 
                        
  • 📅 Agenda (se sua clínica usar)
    • 
                        
  • 📑 Orçamentos
    • 
                        
  • 🦷 Odontograma (dentro do orçamento)
    • 
                        
  • 💰 Comissões
    • 
                        
  • 🩺 Atendimento
    • 
                        
  • 🏭 Produção Protética (Fluxo de custódia com QR Code)
    • 
                        
  • 📣 Marketing (crie campanhas e recupere pacientes)
    • 
                        
  • 🔐 Gerenciar usuários
    • 
                        
  • 🕵️ Auditar Cancelados
    • 
                    

                    
Obs.: o cadastro de novas clínicas é restrito ao dono do sistema (SuperAdmin) 🔒


                    
3) Configurar o logotipo da clínica 🏥🖼️

                    
    
                        
  1. Vá em Configurações → Minha Clínica 🏠🏥
    2. 
                        
  2. Confira se os dados estão corretos ✏️
    3. 
                        
  3. Em Logotipo (Arquivo):
                            
      
                                
    • selecione a imagem 📷
      • 
                                
    • aguarde aparecer a prévia 👀
      • 
                            
    
                        
    4. 
                        
  4. Clique em Salvar 💾✅
    5. 
                    

                    
Pronto! Seu logotipo passa a aparecer no sistema e em relatórios 📄✨


                    
4) Cadastrar seus primeiros dentistas/profissionais 👩‍⚕️🧑‍⚕️

                    
    
                        
  1. No menu, abra Profissionais 👥
    2. 
                        
  2. Clique em Novo Profissional ➕
    3. 
                        
  3. Preencha:
                            
      
                                
    • 🧾 Nome
      • 
                                
    • 🧠 Especialidade (se tiver)
      • 
                                
    • 📞 Contato (opcional)
      • 
                            
    
                        
    4. 
                        
  4. Clique em Salvar ✅
    5. 
                    

                    
Dica: cadastre também recepção/auxiliar se vocês quiserem controlar permissões por usuário 👩‍💼👨‍💼


                    
5) Criar usuários (logins) da equipe 👤🔑

                    
Para cada pessoa que vai usar o sistema (dentista, recepção, auxiliar):

                    
    
                        
  1. Abra Usuários (Admin) 👤⚙️
    2. 
                        
  2. Clique em Novo Usuário ➕
    3. 
                        
  3. Informe:
                            
      
                                
    • 📧 E-mail do colaborador
      • 
                                
    • 🔑 Senha inicial (você define)
      • 
                                
    • 🏷️ Perfil (Dentista / Recepção / Auxiliar / Admin)
      • 
                            
    
                        
    4. 
                        
  4. Clique em Salvar Usuário ✅
    5. 
                    

                    
    
                        
  • ✅ O usuário já fica ativo (não precisa confirmação por e-mail) ⚡
    • 
                        
  • 🔐 O perfil define o que a pessoa pode ver e fazer
    • 
                        
  • 🛡️ Você pode ajustar permissões conforme sua rotina
    • 
                    


                    
6) Configurar serviços (tabela de preços) 🧾💰

                    
    
                        
  1. Abra Serviços/Estoque 🧰
    2. 
                        
  2. Clique em Novo Serviço ➕
    3. 
                        
  3. Preencha:
                            
      
                                
    • 📝 Descrição (ex.: “Restauração”, “Canal”, “Coroa”)
      • 
                                
    • 💵 Valor
      • 
                                
    • 🦷 Se exige seleção por dente/elemento (quando aplicável)
      • 
                            
    
                        
    4. 
                        
  4. Clique em Salvar ✅
    5. 
                    

                    
Dica: se o serviço for “por dente”, marque a opção de cálculo/elemento 🦷📌


                    
7) Cadastrar o primeiro paciente 👤📇

10) Dúvidas comuns ❓

                    
“Meus dados ficam separados de outras clínicas?” 🛡️

                    
Sim ✅ Cada clínica trabalha dentro do seu próprio ambiente (empresa). Seus usuários, pacientes, orçamentos e campanhas ficam isolados.

                    
“Eu consigo cadastrar novas clínicas?” 🏢

                    
Não 🚫 Isso é restrito ao SuperAdmin do sistema.

                    
“Esqueci a senha de um usuário” 🔑

                    
Crie uma nova senha pelo módulo Usuários (Admin) ou peça suporte ao responsável do sistema 🛠️
